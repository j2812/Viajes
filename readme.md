## Travel 4 World

Tener las siguientes herramientas instaladas 

1. composer
2. nodejs
3. npm o yarn

~~~bash
$ git clone https://github.com/j2812/Viajes.git
$ cd Viajes
$ composer install
$ cat env.example > .env
$ php artisan key:generate
$ npm install
~~~
